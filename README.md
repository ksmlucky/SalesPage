# SalesPage
Python, MongoDB, Flask, html을 사용하여 만든 상품 판매 페이지입니다.   
[제주도 오르골](http://startupchild.shop/)

정말 간단한 웹페이지임에도 불구하고 Mongodb와 Flask, AWS를 처음 사용해 보았기 때문에 많은 시행착오가 있었습니다.
